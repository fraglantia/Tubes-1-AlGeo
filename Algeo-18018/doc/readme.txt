folder bin berisi Java bytecode (.class)
folder src berisi source code program (.java)

Pada tugas ini kami membuat program untuk mengolah matriks seperti menyelesaikan SPL, menghitung determinan, mencari matriks balikan,
mencari matriks kofaktor, mencari matriks adjoin, serta interpolasi polinom. Program ini dibuat dengan bahasa Java.

Cara menjalankan program ini dari terminal adalah dengan masuk ke folder bin lalu ketikkan java DriverMatriks.
